What's new in HTML5?

Cute puppies

HTML 5 recognizes that webpages have a structure, just like books and other XML documents have a structure. In general, webpages have navigation, body content, sidebar content, headers, footers, and other features.

HTML 5 New Form Types

HTML 5 supports all the standard form input types, but it adds a few more: datetime, datetime-local, date, month, week, time, number, range, email, url.
Created at: 5/9/2018

Why new features?

HTML 5 was developed to help web application developers, so there are a lot of new features that make it easy to create dynamic HTML pages.
